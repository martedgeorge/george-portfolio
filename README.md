# George Marte - Portfolio Website

**Redesign of georgedmarte.com from the ground up**

A premium, art-directed portfolio website showcasing graphic design and creative direction work. Built with Next.js, Tailwind CSS, and Framer Motion for award-winning quality motion design and interactions.

## Vision

- **Bold & Memorable** – Separates from generic portfolio templates
- **Experimental & Usable** – Creative interactions without sacrificing clarity
- **Premium Quality** – Inspired by award-winning Awwwards websites
- **Art Direction Focus** – Tells stories through visual design

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Language**: TypeScript
- **State Management**: Zustand

## Color Palette

- Electric Purple: `#6B3FA0`
- Bright Orange: `#FF6A00`
- Deep Navy: `#0F172A`
- Charcoal: `#111827`
- Off-White: `#F8F8F8`

## Project Structure

```
src/
├── app/              # Next.js app router
├── components/       # Reusable components
├── layouts/          # Page layouts
├── hooks/            # Custom React hooks
├── utils/            # Utility functions
├── types/            # TypeScript definitions
├── styles/           # Global styles
└── content/          # Static project content
```

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Homepage** – Hero, featured work, primary navigation
- **Work** – Project showcase with custom layout
- **Project Pages** – Case studies with scroll-triggered animations
- **About** – Personal story and creative philosophy
- **Contact** – Simple, strong call-to-action

## Design Principles

✓ Work remains the hero
✓ Intentional motion, not excessive
✓ High contrast, editorial layouts
✓ Mobile-first responsive design
✓ SEO optimized
✓ Accessibility best practices
